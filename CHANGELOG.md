# Changelog

## [0.1.0] - 2026-03-21

### Added

- File path autocomplete triggered by `@` character
- Works in all file types including unsaved/untitled files
- Configurable trigger character
- Three path styles: relative-to-workspace, relative-to-file, absolute
- Customizable ignored patterns (node_modules, .git, etc.)
- Optional directory path suggestions
- File icon support in completion list
- Respects .gitignore by default
